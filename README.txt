High Noon
---------

Developed using keyboard and VGA monitor interaction resources 
provided by Professor Steven Engels, UofT DCS, for CSC258.

Run on Altera FPGA, model 5CSEMA5F31C6, using Quartus software.

See DemoVideo.mp4 for demonstration of a full gameplay loop.

Running on another PC may require validation that:
*Correct pin assignment file (DE1_SoC.qsf) has been imported
*All project files have been imported correctly
*FPGA is of correct (Altera 5CSEMA5F31C6) or compatible model
